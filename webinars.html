BART Webinars

We are hosting a series of webinars for the BART project. The purpose of the webinars is to demonstrate specific components of the toolbox, starting from basic installation and use all the way through building new MRI reconstruction algorithms.

Day 1

  • Introduction: Compilation and initial setup
  • Where to find documentation, examples, and help
  • Discussion of file format and dimensions
  • Working with Python / Matlab wrappers and command-line commands

Hands-on examples:

  • MRI pre-processing
  • Coil sensitivity calibration
  • Parallel imaging and compressed sensing reconstruction
  • MRI simulation
  • Non-Cartesian reconstruction
  • Subspace reconstruction
  • MRI post-processing

Day 2

  • Introduction to the C-programming library
  • Set up a build environment
  • How to run unit tests
  • Add command-line parameter to existing tool
  • Build a basic BART tool and unit test
